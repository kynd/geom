UI Component Guide

Canonical components for all Geom demos. One layout pattern — the top bar — scales from a single play button up to a full control strip. All demos use the same token sizes, fonts, and colour values.

Tokens
TokenValueUsage
--h-bar28 pxAll controls inside the bar (buttons, selects)
--h-play48 pxPlay button — always; 36 px inside the bar only
--r-bar14 pxPill radius for bar buttons
--r-sq4 pxSelect, panel rows
--font-uiSoraAll labels, buttons, selects
--font-monoGoogle Sans CodeNumeric readouts, function signatures, timestamps
--sz-sm10 pxControls, labels, values
--sz-lg12 pxShape name in bottom-left label

Both fonts must be loaded in every demo. AA button always defaults off — no active class, u_ssaa: 0.

Layout pattern A — Top bar

The only layout pattern. Scale from a single play button up to a full strip of controls. Bar is 64 px tall, controls vertically centred (align-items: center), top gradient scrim. Play button sits at the far right. FPS meter is to the left of play.
